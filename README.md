# Daily Reflection Tree Assignment

## Overview
This project implements a deterministic decision tree for daily reflection and an optional AI agent.

## Part A: Decision Tree
A structured system to analyze daily events and improve decision-making.

## Part B: AI Agent
A simple Python-based agent that interacts with the user and provides reflections.

## Guardrails
Implemented to avoid incorrect or misleading outputs.

## How to Run
1. Install Python
2. Run:
   python agent.py
